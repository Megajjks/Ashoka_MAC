# Navbar

This component defines the style and structure of the user's navbar.

## Props

| Props  | Type    | Description                                    | required |
| ------ | ------- | ---------------------------------------------- | -------- |
| goback | boolean | all data of commitments comming from a request | no       |

## Implementation

```jsx
  //with button back
  <Navbar
		goback=true
  />

  //without button back
	<Navbar/>
```
